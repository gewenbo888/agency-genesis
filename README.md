# Agency Genesis · 主体性创世

A philosophical operating system for the genesis of agency: the transition from being shaped by the world toward consciously shaping oneself. Bilingual (EN / 中文), light/dark, editorial-philosophical aesthetic with Bodoni Moda + Lora + IBM Plex Mono. Ten chapters: hero plate of the conditioning matrix (radial), eight shaping forces with conditioning loop, four observer positions + 40-prompt deck, dopamine-ladder + 5-gate compulsion-to-perception protocol, five identity rewrites + voting model, uncertainty band + four integration stages, four environment redesigns, three orders of effects + three distortions, eight-axis values compass, AI Agency Guide, and a 36-month evolution dashboard.

## Stack

Plain HTML, CSS, vanilla JavaScript. No build step. Bilingual via `[data-lang]` + CSS attribute selectors. Light/dark via `[data-theme]`. Editorial-philosophical aesthetic with Bodoni Moda (display, italic-heavy), Lora (body), IBM Plex Mono. All diagrams are inline SVG.
